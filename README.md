# What Makes People Happy? — WHR 2023 Poster

Data-science poster for the University of Hohenheim, analysing which
socioeconomic factors correlate most strongly with subjective well-being,
based on the World Happiness Report 2023.

Author: Mehmet Akif Coskun (Bachelor Business Informatics).

## View the poster

Open `poster_new.pdf`.

## Re-render from source

R packages required: `rmarkdown`, `knitr`, `ggplot2`, `dplyr`, `readxl`.
Also needs a working LaTeX installation (`pdflatex`) and Pandoc.

```bash
Rscript -e 'rmarkdown::render("poster_new.Rmd")'
```
